= XMind Portable For Windows =

To enable launching this XMind Portable Version by double click on a '.xmind'
file (i.e. to associate '.xmind' files with this XMind.exe), please execute
'Associate_XMind_File.bat' by double click on it.

Please remember that, whenever this XMind Portable Version is moved to another
location, you must manually execute 'Associate_XMind_File.bat' again to make
sure the Windows registry be aware of the new location.

To use those custom fonts carried along this portable package, please execute
'Install_Custom_Fonts.CMD' by double click on it. This process only needs to be
executed for once in a single Windows system.
